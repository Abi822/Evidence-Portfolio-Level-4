COMP4206 Web Design Essentials

Type of Assessment

  1. Copyright Web page
  2. Critique Web page
  3. Internet Basics Web Page
  4. Create a Web Site


Learning outcomes assessed

  • Is Demonstrate knowledge and understanding of web design concepts.
  • Define user experience requirements including: forms that allow user entry, validation, restrict info with drop down list, usability by others, and accessibility for end-users.
  • Create and verify relevant documentation including: user and technical.

Instructions

Part A, B and C – Theory questions

Part A, B, and C are to be done using a Microsoft Word or Google Docs. There will be lots of answers which require least a paragraph on each point, word answers are not accepted. This is an individual assignment.

You have to produce an assignment that shows the level of your understanding of copyright, and plagiarism, critique of screen design fundamentals in relationship to web site design and show your knowledge of Internet basics.

Part D – Website

After you have completed your Business Plan, you have to make a website based on this business. There needs to be at least five pages using HTML5 and CSS.
